# Attunehq Attune

## How do I install these formulae?

`brew install attunehq/attune/<formula>`

Or `brew tap attunehq/attune` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "attunehq/attune"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
